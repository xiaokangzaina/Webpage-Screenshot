# 网页实时获取截图

AstrBot 网页截图插件。支持按群聊定时截取网页并推送，也支持在目标会话内发送“状态”立即获取截图。

## v1.3.0 — UI 大改

本次更新重点重写了插件 Web 配置页，让网页截图任务配置从简易表单升级为更完整的控制台式界面。

### UI/交互重构

- 全面重写前端配置页，采用控制台式布局。
- 顶部加入状态概览：群聊总数、已配置群聊、当前间隔、当前网页。
- 群聊列表改为卡片式侧栏，支持搜索、头像、最近配置置顶。
- 左侧群聊列表在桌面端保持固定，群列表内部独立滚动。
- 右侧配置页自然滚动，不再裁切底部配置项。
- 配置项分为「基础任务」「消息行为」「渲染参数」三块。
- 新增浏览器风格实时预览卡，显示 URL、任务名、截图文字、尺寸、等待/超时参数。
- 主题样式重做，支持浅色/深色/自动主题。
- 顶部标题栏压缩，减少无效留白。
- 状态卡和操作按钮布局优化，保存/清空按钮移动到状态栏右侧。
- 移动端自动切换单列布局。

### Bug 修复

- 修复页面主题只保存在 localStorage 导致刷新或换环境后可能丢失的问题。
- 新增后端主题保存接口，主题会持久化到插件配置：

```json
"page_theme": "auto"
```

可选值：

```text
auto / light / dark
```

### 后端兼容

- 保持原有群截图任务接口不变。
- 新增接口：

```text
POST /astrbot_plugin_webpage_screenshot/settings/theme
```

用于保存配置页主题偏好。

## 功能特性

- 按群聊配置独立网页截图任务。
- 支持群聊列表同步和搜索。
- 支持最近配置群聊置顶。
- 支持每个群独立配置截图 URL、推送间隔、消息文本、窗口尺寸、等待时间、超时时间。
- 支持失败通知。
- 支持手动发送“状态”立即截图。
- 配置保存后会唤醒等待中的定时任务，新间隔立即生效。

## 单群配置项

| 配置项 | 说明 |
|---|---|
| 启用该群截图任务 | 是否启用该群定时截图。 |
| 任务名称 | 用于日志和消息模板。 |
| 网页地址 | 要截图的完整 URL。 |
| 推送间隔分钟 | 定时截图间隔。 |
| 发送截图文字 | 是否发送截图附带文字。 |
| 截图文字 | 支持 `{name}` 和 `{url}`。 |
| 引用和艾特发送人 | 手动“状态”截图时引用并艾特。 |
| 失败通知 | 定时截图失败时通知该群。 |
| 浏览器宽度 | 截图浏览器宽度。 |
| 浏览器高度 | 截图浏览器高度。 |
| 页面等待秒数 | 页面加载后等待时间。 |
| 页面超时秒数 | 页面最大加载等待。 |

## 使用方法

1. 打开 AstrBot Dashboard。
2. 进入插件管理。
3. 打开“网页实时获取截图”的配置页面。
4. 左侧选择目标群聊。
5. 填写网页地址。
6. 开启“启用该群截图任务”。
7. 设置推送间隔、消息文本和截图参数。
8. 点击“保存该群配置”。

在已配置的目标会话内发送：

```text
状态
```

可立即获取一次截图。

## 依赖安装

```bash
pip install -r requirements.txt
playwright install chromium
```

## 配置与数据文件

插件会使用以下文件保存数据：

```text
data/webpage_screenshot_tasks.json
```

用于保存群截图任务。

```text
data/group_config_touch_times.json
```

用于记录群配置更新时间，方便最近配置群聊置顶。

```text
data/config/astrbot_plugin_webpage_screenshot_config.json
```

用于保存普通配置和页面主题偏好。

## 历史版本

### v1.2.0

- 保存配置后立即唤醒等待中的定时任务，用新间隔重新对齐，无需等上一轮结束。
- 引用和艾特发送人：优先从分群任务配置读取，不再依赖已移除的全局字段。
- 发送截图文字、失败通知同理：优先从分群任务配置读取。
- 普通插件配置页仅保留默认截图参数和高级设置。

### v1.1.1

- 前端配置不再写入 AstrBot 普通配置页的 `tasks` 模板列表。
- 前端配置改为写入插件独立数据文件：`data/webpage_screenshot_tasks.json`。
- 避免普通配置页 `template_list` 因模板标识兼容问题提示“找不到对应模板”。
- 插件运行时会自动读取独立数据文件中的群截图任务。

### v1.1.0

- 新增 Web 前端配置页。
- 左侧显示 QQ 群列表，进入页面自动同步。
- 最近配置过的群聊自动置顶。
- 点击群聊后配置该群的截图任务。
- 原全局配置已融入单群配置，每个群都可以独立设置。
- 米黄色主题 UI。
- 群列表超出高度时自动滚动。

## 注意事项

- 发布包不会包含本地运行数据和真实群号。
- API Key、Token、Cookie 不需要配置。
- 如网页加载较慢，请适当调大“页面等待秒数”。
- 首次安装 Playwright 后需要执行 `playwright install chromium`。
